
# Build a Spreadsheet

A webapp that uses functional programing to create a spreadsheet. Uses array manipulation and regex filtering to allow it to make calculations.

Performs the following functions:
```
sum, 
average, 
median, 
even, 
someeven, 
everyeven, 
firsttwo,
lasttwo,
has2,
increment,
random,
range,
nodupes
```
